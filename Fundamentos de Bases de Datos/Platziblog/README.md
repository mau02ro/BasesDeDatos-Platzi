- **Post**

  - Titulo
  - Fecha de publicación
  - Contenido
  - Estatus
  - Etiquetas (multivaluado)
  - IdPost

- **Usuarios**

  - Login

  - Nickname
  - Email
  - IdUsuario

- **Comentarios**

  - Contenido
  - Fecha de publicación
  - IdComentario
  - IdUsuario (clave foránea)

- **Categorías**
  - Npmbre de Categoría
  - IdCategoria
